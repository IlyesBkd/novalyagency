# Campagne Google Ads — « agence + ville »

Stratégie minimaliste : **1 seule campagne**, **1 seul groupe d'annonces**, **1 seule annonce**, et des mots-clés exclusivement composés de `agence` + un nom de ville française.

---

## 1. Structure

- **Compte** : Noval Agency
- **Campagne** : `Search - Agence - Villes FR`
- **Groupe d'annonces** : `Agence + Ville`
- **Annonce** : 1 seule annonce responsive (RSA)

---

## 2. Paramètres de la campagne

| Paramètre | Valeur |
|---|---|
| Type | Search (Réseau de Recherche uniquement) |
| Réseau Display | **Désactivé** |
| Partenaires de recherche | **Désactivé** |
| Zone géographique | France |
| Langue | Français |
| Stratégie d'enchères | **Maximiser les clics** (avec CPC max plafonné) ou **CPC manuel** |
| CPC max suggéré | 1,50 € (à ajuster) |
| Budget quotidien | 15 € (à ajuster) |
| Calendrier | 24/7 (à optimiser après 2 semaines) |
| Rotation des annonces | Optimiser |
| Appareils | Tous (ajustement -20 % tablette au démarrage) |

---
